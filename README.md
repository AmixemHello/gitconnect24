# GitConnect24
To install the service, execute:
```bash
git clone gitconnect24
cd "service/main.html"
exec "sh start /main"
```
then start http://localhost:8080
## Requirements
You will need:
* Node.js
* Python 2.4.6 or later
* Windows 8.0 or later/Android 6.3 or later
* Terminal (only needed for Android)
* Internet
