# How to use it?

First, you need to know what Android versions are supported.

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :x: |
| < 4.0   | :x:                |

Also, you need to know where you install GitConnect24.


Like in a GitConnect24 folder.
